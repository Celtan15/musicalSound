Aquí aprenderás el panel de opciones de FL Studio

Panel de opciones

En esta sección encontrarás las opciones que permitirán crear, exportar nuevos proyectos, así como también agregar nuevos plugin y patrones necesarios para tus proyectos, entre muchas otras funciones que deseas realizar. A continuación explicaremos cada una de esas opciones

¡Hola!

En la esquina superior izquierda podrás encontrar el siguiente panel

File (Archivo)

Una de las primeras opciones que encontrarás es "Open" que te permitirá abrir un proyecto ya creado, que se encuentra en tu dispositivo, De esta misma manera con la opción "Save as", con la que puedes guardar tus proyectos

Tus proyectos se guardarán con la extensión ".flp"

Una de las opciones importantes es importar que permite traer diferentes tipos de archivos como audio (mp3, WAV, entre otros), archivos MIDI. Con la opción exportar permite guardar y convertir tus proyectos en diferentes formatos.

Puedes exportar tus proyectos en formatos como: WAV, MP3, OGG, MIDI, M4A.

Add (Agregar)

En esta opción podrás agregar nuevos plugins que necesites en tus proyectos, además de los que ya vienen por defecto. Se debe tener en cuenta que estos deben guardarse en la carpeta "plugins", que se crea al instalar FL Studio

Podrás encontrar plugins en la sección Tips y Material de apoyo

Patterns (Patrones)

Con esta opción podrás agregar patrones en los instrumentos musicales que requieras, con el fin de crear secuencias, acordes y ritmos. Aquí podrás encontrar patrones por defecto o también crear uno nuevo.

Puedes crear tus propios patrones desde el piano roll.

View (Ver)

Con la opción "VIEW" podrás visualizar las ventanas y elementos que se encuentran en FL Studio, además de tener la opción de cerrarlas. Podrás organizar las ventanas como lo prefieras y distribuirlas de mejor manera en la pantalla.

Options (Opciones)

Permite configurar de manera personalizada tu entorno de trabajo, de manera general o de herramientas específicas, como el MIDI, audio, también configurar el proyecto, administrar los plugins.

Esta opción te permite cambiar el idioma, el tema, la apariencia que prefieres para tu mejor visualización.

Todas las configuraciones que desees realizar las encontrarás aquí.

Tools (Herramientas)

En esta opción podrás encontrar herramientas y funciones útiles para la producción y edición de audio, como "Riff machine" que te permite generar nuevos patrones y ritmos de manera automática. "Macros" es una opción que puedes utilizar para limpiar tu proyecto de datos que no están siendo utilizados.

Riff machine es una herramienta que sirve como inspiración para crear nuevos patrones

Con "Control creator",puedes personalizar los diseños de los controles MIDI virtuales como perillas, botones.
